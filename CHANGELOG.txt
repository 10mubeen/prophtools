CHANGELOG
=========

09/11/2017 - Prophtools v1.1.

* TXT file support (explained now in the documentation).
* GEXF file support based on simlegexf python library (https://github.com/demux/simplegexf) and networkx (https://networkx.github.io/).
* Group attribute on GEXF nodes to define heterogeneous networks.
* Conversion GEXF to .mat files for compatibility with previous versions of prophnet/prophtools.
* Csv output for prioritization results.
* labels_as_ids option to use label in input files as identifiers.

* BETA: Memsave mode for very large networks.
